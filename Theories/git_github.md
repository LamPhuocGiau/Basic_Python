### Setup

- Install git.

- Check git.

```
git version
```

- Create github account.

- Connect git and git hub.

```
git config --global user.name “Giau_Lam”
git config --global user.email “lamphuocgiau86@gmail.com”
```
- Check git.

```
git config --list
```
